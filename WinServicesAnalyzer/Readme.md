[WinServicesAnalyzer]  

List of Windows services information    

- OS: Windows 10 64Bit  
- Source: %SystemRoot%\System32\config\SYSTEM  
- Usage:  
`WinServicesAnalyzer.exe -f SYSTEM` //Analyze a file  
`WinServicesAnalyzer.exe -f SYSTEM -local` //Time value is displayed in local time

[MftExtractor]  

Export $MFT of NTFS  

- OS: Windows 10 64Bit (**Need to Run as Administrator**)  
- Source: $MFT  
- Usage: Input NTFS drive letter (ex. C:) and output path  
`MftExtractor.exe -d C: -o C:\OutputFolder`

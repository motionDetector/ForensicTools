[WinAccountAnalyzer]  

List of Windows account information           

- OS: Windows 10 64Bit  
- Source: %SystemRoot%\System32\config\SAM  
- Usage:  
`WinAccountAnalyzer.exe -f SAM` //Analyze a file   
`WinAccountAnalyzer.exe -f SAM -local` //Time value is displayed in local time

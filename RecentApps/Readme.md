[RecentAppsView]  

List of programs and items  

- OS: Windows 10 64Bit  
- Source: HKEY_CURRENT_USER\SOFTWARE\Microsoft\Windows\CurrentVersion\Search\RecentApps  
- Usage:  
`RecentAppsView.exe` //print results  
`RecentAppsView.exe -csv` //export csv file

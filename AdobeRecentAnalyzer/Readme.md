[AdobeRecentAnalyzer]  

List of Adobe files  

- OS: Windows 10 64Bit  
- Source: %UserProfile%\NTUSER.DAT    
- Usage:  
`AdobeRecentAnalyzer.exe -f NTUSER.DAT` //Analyze a file  
`AdobeRecentAnalyzer.exe -f NTUSER.DAT -local` //Time value is displayed in local time

[OfficeRecentView]  

List of recent office files 

- OS: Windows 10 64Bit
- Source: HKEY_CURRENT_USER\SOFTWARE\Microsoft\Office\\*
- Usage:  
OfficeRecentView.exe //print info  
OfficeRecentView.exe -csv //export csv file

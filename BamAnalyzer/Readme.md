[BamAnalyzer]  

List of executed programs  

- OS: Windows 10 64Bit  
- Source: %SystemRoot%\System32\config\SYSTEM  
- Usage:  
`BamAnalyzer.exe -f SYSTEM` //Analyze a file  
`BamAnalyzer.exe -f SYSTEM -local` //Time value is displayed in local time

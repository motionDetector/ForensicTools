[AdobeRecentView]  

List of recent adobe files  

- OS: Windows 10 64Bit
- Source: HKEY_CURRENT_USER\SOFTWARE\Adobe\\*  
Usage:  
AdobeRecentView.exe //print info  
AdobeRecentView.exe -csv //export csv file

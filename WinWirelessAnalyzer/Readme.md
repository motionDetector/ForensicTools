[WinWirelessAnalyzer]  

List of connected wireless networks  

- OS: Windows 10 64Bit  
- Source: %SystemRoot%\System32\config\SOFTWARE   
%SystemRoot%\System32\config\SYSTEM  
- Usage:  
`WinWirelessAnalyzer.exe -f SOFTWARE` //Analyze a file  
`WinWirelessAnalyzer.exe -f SYSTEM` //Analyze a file  
`WinWirelessAnalyzer.exe -f SYSTEM -local` //Time value is displayed in local time

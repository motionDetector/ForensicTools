[RecentDocsAnalyzer]  

List of recent apps and items  

- OS: Windows 10 64Bit  
- Source: %UserProfile%\NTUSER.DAT  
- Usage:  
`RecentDocsAnalyzer.exe -f NTUSER.DAT` //Analyze a file  
`RecentDocsAnalyzer.exe -f NTUSER.DAT -local` //Time value is displayed in local time

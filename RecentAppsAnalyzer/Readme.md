[RecentAppsAnalyzer]  

List of recent apps and items  

- OS: Windows 10 64Bit  
- Source: %UserProfile%\NTUSER.DAT  
- Usage:  
`RecentAppsAnalyzer.exe -f NTUSER.DAT` //Analyze a file  
`RecentAppsAnalyzer.exe -f NTUSER.DAT -local` //Time value is displayed in local time

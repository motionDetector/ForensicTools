[DeviceInstallLogView]  

Export list of installed(connected) device  

- OS: Windows 10 64Bit  
- Source: C:\Windows\INF\setupapi.dev.log  
- Usage:   
`DeviceInstallLogView.exe -f setupapi.dev.log`

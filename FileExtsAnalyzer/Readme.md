[FileExtsAnalyzer]  

List of ext and OpenWith programs   

- OS: Windows 10 64Bit  
- Source: %UserProfile%\NTUSER.DAT  
- Usage:  
`FileExtsAnalyzer.exe -f NTUSER.DAT` //Analyze a file  
`FileExtsAnalyzer.exe -f NTUSER.DAT -local` //Time value is displayed in local time
